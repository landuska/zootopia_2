# Zootopia Animals Project

This project retrieves information about animals and displays it using Python.

## Features

- Retrieves animal data from an external API
- Displays useful information about animals
- Simple command-line interface
- Easy to run and modify

## Installation

1. Clone the repository:

git clone https://github.com/landuska/zootopia_2.git

2. Navigate to the project folder:

cd zootopia_2

3. Install the required dependencies:

pip install -r requirements.txt

## Usage

Run the program with the following command:
python animals.py

The program will fetch animal data and display the results.

## Project Structure

- `animals.py` – main program that runs the project
- `data_fetcher.py` – retrieves animal data from the API
- `animals_template_2.html` – HTML template used to generate the webpage
- `animals_2.html` – generated HTML file with the animal information
- `.env` – environment variables (for example API keys)
- `requirements.txt` – project dependencies
- `README.md` – project documentation
